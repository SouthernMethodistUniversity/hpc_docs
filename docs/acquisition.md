# Software Acquisition Process

We provide a multitude of software options on both M3 and the SuperPOD. However, if you find that you need a specific software for your research, we follow the process outlined below.

## Open-Source Software

We encourage users to build their own software if possible but if a user needs assistance or feels the software would be best suited for a system-wide installation, please email help@smu.edu with [HPC] in the subject line.

## Licensed Software

If you wish to purchase a site license or user-specific license for your research, we follow the following procedure:

- Create a ticket by emailing help@smu.edu with any relevant information regarding the software
- OIT Research Services will contact you for a consultation. Our primary goal will be to determine the compatibility of the software on our HPC system
- Our system administrators will handle the requisition of the software with SMU Purchasing
- OIT Research Services will install and validate the software is properly configured and communicate to you once it is ready for use.

## Funding for Software

There is a small amount of funding available for software that has broad impact to the HPC user community which will be determined by HPC Governance. If your software is more specific to your school/department/research group, we encourage you to reach out to your Faculty Technology Council representative, Dean, or chair to explore funding options.
